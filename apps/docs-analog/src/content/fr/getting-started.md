---
title: Premiers pas
description: Démarrez une nouvelle application Analog en moins d'une minute.
---

Le moyen le plus rapide de commencer est le template officiel.

```bash
npm create analog@latest my-app
cd my-app
npm install
npm start
```

:::info
Node 22 ou 24 est requis. Les versions antérieures ne sont pas prises en charge.
:::
