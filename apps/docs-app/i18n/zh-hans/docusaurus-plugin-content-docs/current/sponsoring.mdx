# 赞助

Analog 是一个 基于 MIT 协议的开源项目并且完全免费使用。维护该项目所需的持续努力得到了我们赞助商的资金支持。

你可以 [在 GitHub 上赞助 Analog](https://github.com/sponsors/brandonroberts).

### 商业/企业赞助

支持 OSS 可以提升您的品牌声誉，这对于任何与开发人员互动的公司来说都是一项重要资产。

如果您正在使用 Analog 构建商业产品，赞助其开发也有助于您的利益。它可以确保您的产品所依赖的项目保持健康并得到积极维护。它还为您的品牌在社区中提供了积极的地位，也使寻找和招募 Angular 开发人员变得更加容易。赞助还可以建立品牌知名度并提高转化率。

### 以个人身份赞助

如果您是个人用户，并享受使用 Analog 带来的生产力，请考虑赞助我以表示感谢 - 比如偶尔请我喝咖啡。如果您不想每月定期赞助，Github 还支持一次性捐赠！

你还可以尝试说服您的雇主以企业形式赞助 Analog。这可能并不容易，但企业赞助通常比个人捐赠对 OSS 项目的可持续性产生更大的影响，因此如果您成功了，您将对我们帮助更大。

如果您对赞助有任何疑问，请联系我们：sponsor@analogjs.org。
