# Patrocinio

Analog es un proyecto de código abierto con licencia MIT y completamente gratuito para usar. El esfuerzo continuo necesario para mantener este proyecto está respaldado por el apoyo financiero de nuestros patrocinadores.

Puedes [patrocinar a Analog en GitHub Sponsors](https://github.com/sponsors/brandonroberts).

### Patrocinios empresariales o corporativos

Apoyar el software de código abierto (OSS) mejora la reputación de tu marca, lo cual es un activo importante para cualquier empresa que interactúa con desarrolladores.

Si estás utilizando Analog para construir un producto que genera ingresos, patrocinar su desarrollo también beneficia tus intereses. Asegura que los proyectos en los que se basa tu producto permanezcan saludables y activamente mantenidos. También posiciona positivamente a tu marca en la comunidad y facilita encontrar y reclutar desarrolladores de Angular. El patrocinio también construye reconocimiento de marca y mejora la conversión.

### Patrocinio como individuo

Si eres un usuario individual y has disfrutado de la productividad al usar Analog, considera patrocinarme como una muestra de agradecimiento, como si me compraras un café de vez en cuando. Si no te sientes cómodo con un patrocinio recurrente mensual, GitHub también admite donaciones únicas.

También puedes intentar convencer a tu empleador de patrocinar Analog como negocio. Esto puede no ser fácil, pero los patrocinios empresariales suelen tener un impacto mucho mayor en la sostenibilidad de los proyectos OSS que las donaciones individuales, así que nos ayudarás mucho más si tienes éxito.

Si tienes alguna pregunta sobre patrocinios, por favor contáctanos a través del email sponsor@analogjs.org.
