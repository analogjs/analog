Analog is a fullstack meta-framework for building applications and websites with [Angular](https://angular.dev).

Similar to other meta-frameworks such as Next.JS, Nuxt, SvelteKit, Qwik City, and others, Analog provides a similar experience, building on top of Angular.

## Features

- Supports Vite/Vitest/Playwright
- Server and deployment integrations powered by [Nitro](https://nitro.unjs.io)
- [File-based routing](/features/routing/overview)
- [Server-side data fetching](/features/data-fetching/server-side-data-fetching)
- [Support for using Markdown as content routes](/features/routing/content)
- [API/server routes](/features/api/overview)
- Hybrid [SSR](/features/server/server-side-rendering)/[SSG support](/features/server/static-site-generation)
- Supports Angular CLI/[Nx workspaces](/integrations/nx)
- Supports [Angular components](/packages/astro-angular/overview) with Astro
